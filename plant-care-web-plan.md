# 초록 돌봄방: 함께 키우는 식물 다마고치 웹 기획안

## 1. 프로젝트 개요

학교 식물 재배 프로그램을 기반으로, 여러 학생이 QR코드를 통해 웹에 접속하여 식물을 함께 돌보는 참여형 웹 서비스를 제작한다.
이 웹은 식물의 키를 cm 단위로 기록하는 관찰일지가 아니라, **물·햇빛·관찰·응원 메시지 등 돌봄 행위가 웹 속 식물 캐릭터의 상태와 분위기로 반영되는 식물 다마고치형 서비스**이다.

## 2. 핵심 콘셉트

* 실제 식물 옆에 QR코드를 배치한다.
* 학생들은 QR을 통해 웹에 접속한다.
* 학생들은 식물을 보고 간단한 돌봄 행동을 선택한다.
* 웹 속 식물 캐릭터는 학생들의 행동에 따라 표정, 배경, 색감, 효과가 달라진다.
* 식물의 성장은 단순한 크기보다 **함께 자란 시간, 마지막 돌봄 이후 지난 시간, 오늘 받은 관심 수, 남겨진 마음**으로 표현한다.

## 3. 주요 사용자

* 식물을 직접 키우는 학생
* 식물을 지나가다 구경하는 학교 학생들
* 활동 과정을 확인하는 담당 교사

## 4. 핵심 기능

### 4.1 함께 자란 시간 표시

메인 화면에 식물이 학교에서 함께 자라고 있는 시간을 보여준다.

예시 문구:

* 함께 자란 지 17일 6시간
* 마지막 돌봄 이후 9시간
* 오늘 받은 관심 8개
* 지금까지 남겨진 마음 42개

### 4.2 돌봄 버튼

학생들이 누를 수 있는 버튼을 제공한다.

* 흙 확인했어요
* 햇빛 확인했어요
* 잎을 관찰했어요
* 응원 남기기
* 새 변화를 발견했어요

주의: 실제 “물 줬어요” 버튼은 과습을 방지하기 위해 담당자만 사용할 수 있도록 한다.

### 4.3 인터랙션 효과

학생이 버튼을 누르면 웹 화면에 감성적인 시각 효과가 나타난다.

* 흙 확인: 화분 주변에 작은 반짝임 표시
* 햇빛 확인: 배경에 따뜻한 빛 번짐 효과
* 잎 관찰: 식물 주변에 돋보기 또는 작은 잎사귀 효과
* 응원 남기기: 메시지가 잎사귀 카드로 저장됨
* 새 변화 발견: 식물 주변에 작은 꽃 또는 반짝임 효과

### 4.4 식물 상태 문구

식물의 상태를 딱딱한 관리 용어가 아니라 따뜻한 문장으로 표현한다.

예시:

* 오늘은 누군가의 시선을 기다리는 중이에요.
* 작은 관심이 잎사귀로 남았어요.
* 빛을 확인해줘서 조금 더 따뜻해졌어요.
* 아직 큰 변화는 없지만, 자라는 시간은 쌓이고 있어요.
* 오늘도 천천히 자라고 있어요.

### 4.5 응원 메시지

학생들은 식물에게 짧은 응원 메시지를 남길 수 있다.

조건:

* 30자 이내
* 익명 참여
* 이름, 학번, 반 등 개인정보 수집 금지

메시지 예시:

* 천천히 자라도 괜찮아.
* 오늘 햇빛 좋아 보여.
* 새잎 난 거 축하해.
* 나도 너처럼 조금씩 자라는 중.

메시지는 일반 게시판이 아니라 잎사귀 카드, 물방울 카드, 햇빛 카드 형태로 시각화한다.

## 5. 페이지 구성

### Home

* 식물 캐릭터 표시
* 함께 자란 시간 표시
* 오늘의 상태 문구
* 오늘 받은 관심 수 표시
* 돌봄 버튼 배치

### Care

* 흙 확인
* 햇빛 확인
* 잎 관찰
* 새 변화 발견
* 담당자용 물 주기 기록

### Memory

* 학생들이 남긴 응원 메시지
* 날짜별 돌봄 흔적
* 잎사귀 카드 형태의 메시지 모음

### Growth

* 날짜별 식물 사진
* 함께 자란 시간 기록
* 오늘의 관심 수 변화
* 돌봄 행동 누적 기록

## 6. 디자인 방향

전체 분위기는 따뜻하고 조용한 감성으로 설계한다.

* 컬러: 연두, 초록, 크림색, 연노랑 중심
* 분위기: 귀엽지만 너무 유치하지 않게
* 키워드: 돌봄, 관심, 성장, 시간, 따뜻함
* UI 형태: 둥근 카드, 부드러운 그림자, 식물 일러스트, 잎사귀 모티프
* 버튼 문구는 명령형보다 감성적인 표현 사용

## 8. MVP 구현 범위

1차 버전에서는 아래 기능만 우선 구현한다.

* QR 접속용 반응형 웹페이지
* 식물 캐릭터 메인 화면
* 함께 자란 시간 자동 카운트
* 돌봄 버튼 4개
* 버튼 클릭 시 시각 효과 변화
* 응원 메시지 입력
* 메시지 카드 목록
* 오늘 받은 관심 수 표시
* 날짜별 돌봄 로그 저장

## 9. 최종 목표

이 웹의 목표는 식물을 효율적으로 관리하는 것이 아니라,
학교 안의 여러 학생이 식물을 바라보고 돌본 시간이 웹 안에 따뜻하게 남도록 하는 것이다.
식물은 단순한 재배 대상이 아니라, 학생들의 관심과 애정이 모이는 작은 매개체가 된다.
