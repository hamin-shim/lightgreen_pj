# 초록 돌봄방

학교 식물 돌봄을 위한 GitHub Pages용 정적 웹사이트입니다.

## 포함된 파일

- `index.html` - 홈페이지
- `styles.css` - 디자인 스타일
- `script.js` - 로컬 상호작용 및 상태 저장
- `intro.md` - 기획안 원본
- `plant-care-web-plan.md` - 기획안 사본

## 사용 방법

1. 이 디렉터리를 GitHub 리포지터리로 초기화합니다.
2. `index.html` 파일이 루트에 있어야 합니다.
3. GitHub 리포지터리의 `Settings` > `Pages`로 이동합니다.
4. `Branch`를 `main` 또는 `master`로 설정하고 `/ (root)`를 선택합니다.
5. 저장 후 배포 URL을 확인합니다.

## 기능

- 식물 돌봄 버튼 4개
- 응원 메시지 입력 및 잎사귀 카드 저장
- 오늘 받은 관심 수 자동 계산
- 돌봄 기록 및 성장 기록 뷰
- 브라우저 `localStorage`로 데이터 유지

## 배포

GitHub에 푸시한 후 Pages 배포가 활성화되면 사이트가 자동으로 공개됩니다.
